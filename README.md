**基于Gemini开发的sealdice插件**

灵感来源：智谱大模型 AI Plugin、SillyTravel、SpectreCore (影芯)
（本质上是被智谱模型的呆傻气晕的产物）
（使用了Google ai studio、ChatGPT辅助编码）


**目前此插件实现的功能有：**

1.接入ai模型，自定义prompt进行角色扮演

2.多提示词不同概率回复或@bot回复

3.特殊用户配置，给予不同的user独特身份

4.聊天评分/好感度系统，ai将对user的每一次交互打分，并自主决定好感度提升或下降

5.压力熔断，在短时间内同一人连续输出负面消息时将会提升压力，高压力缩减token或忽视user输入

6.时间感知，ai能够感知当前的时间（此版本中为世界时）
